# 🚀 Crypto Profit Calculator 💰  

A simple **Python (Flask) web app** that calculates crypto trading profits and losses.  
Includes a **Jira project management setup** for tracking Agile tasks.

---

## 🚀 Tech Stack  

- **Backend:** Flask (Python)  
- **Frontend:** HTML, JavaScript  
- **API:** CoinGecko (for live crypto prices)  
- **Project Management:** Jira (CSV issues report included)  

---

## 📌 What is this project?  

This is a simple web app that helps users calculate how much **profit or loss** they made when trading cryptocurrency.  

### 🛠 Inputs:  
1️⃣ **Buy Price** – The price at which you bought the crypto  
2️⃣ **Sell Price** – The price at which you sold the crypto  
3️⃣ **Investment Amount** – The total money you invested  

### ✅ Output:  
The app calculates the **profit or loss** and displays the percentage gain/loss.  

---

## 💻 How does it work?  

- The app is built using **Python (Flask)** to handle backend logic.  
- It uses **CoinGecko API** to fetch live cryptocurrency prices.  
- Users enter their data and click **“Calculate”** to see results instantly.  

---

## 🚀 Why is this project useful?  

✅ Helps **crypto traders** track their profits/losses easily.  
✅ Uses **live market prices** for accurate calculations.  
✅ Very **simple to use** – just enter values and get results!  
✅ Shows **percentage gain/loss** for better understanding.
